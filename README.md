# Dead Island 2 Macro

AutoHotkey v2 macro for Dead Island 2 - Fast Attack & Fast Jump

## Usage

1. Run `DeadIsland2_Macro.exe`
2. Launch Dead Island 2
3. Use hotkeys in-game

## Hotkeys

| Key | Action |
|-----|--------|
| `XButton1` (Mouse Side 1) | Fast Attack |
| `XButton2` (Mouse Side 2) | Big Jump |
| `DEL` | Exit Script |

> [!TIP]
> Hold `CapsLock` for slower attack timing.
